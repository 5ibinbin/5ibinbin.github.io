LeetCode：11.盛最多水的容器

发表于 2019-06-16 | 分类于 LeetCode | 阅读次数 |

题目

盛最多水的容器

描述

给定 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

示例

输入: [1,8,6,2,5,4,8,3,7]
输出: 49

讲解

题目理解：如上图，垂直的两条线段将会与坐标轴构成一个矩形区域，矩形面积越大代表容器可以容纳更多的水，所以此题可理解为求矩形区域面积的最大值

解题思路

方法一：暴力求解

可以通过两次for循环求出所有的可能出现的情况，找出最大值

复杂度分析
  • 时间复杂度：O(n2)，计算所有的n(n-1)/2种情况
  • 空间复杂度：O(1)
方法一：双指针法

将两个指针放到数组的开始和结尾，使用maxArea来存储矩形面积。每循环一次都会更新maxArea，并移动较小数值所对应的指针(为什么移动较小数值的？看这个矩形，在宽度一定(两个指针移动之后的宽度都是-1，所以宽度一定)的情况下，是不是保留高度高的情况下才有可能使面积更大)，最后就得出了最大的面积

复杂度分析
  • 时间复杂度：O(n)，只需要扫描一次
  • 空间复杂度：O(1)
