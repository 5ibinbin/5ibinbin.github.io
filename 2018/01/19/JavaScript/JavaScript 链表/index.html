前言

我们知道数组是一个查找很迅速的数据结构而链表的优势就在于插入和删除很快的一种数据结构，此次将通过实例来实现链表的数据结构。

链表API

  • append(element): 向列表尾部添加一个新的项
  • insert(position, element): 向列表的特定位置插入一个新项
  • remove(element): 从列表中移除
  • removeAt(position): 从列表的特定位置移除一项
  • isEmpty(): 判断列表是否为空
  • size(): 返回链表的元素个数
